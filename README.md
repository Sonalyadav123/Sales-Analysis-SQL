# Sales Analysis Using SQL

## Project Overview

This project analyzes e-commerce sales data using SQL.
The goal is to understand revenue trends, top customers, and product performance.

## Dataset

* File: Day1_Sales_Analysis_Data.csv
* Rows: 1000
* Columns: 11

### Columns Description

| Column          | Description       |
| --------------- | ----------------- |
| OrderID         | Unique order ID   |
| OrderDate       | Date of order     |
| CustomerID      | Unique customer   |
| ProductName     | Product sold      |
| Category        | Product category  |
| Quantity        | Number of items   |
| UnitPrice       | Price per item    |
| DiscountPercent | Discount applied  |
| TotalAmount     | Final order value |
| Region          | Sales region      |
| PaymentMethod   | Payment type      |

---

## Business Questions

1. What is the total revenue?
2. Which product generates the most sales?
3. Who are the top customers?
4. What is the monthly sales trend?
5. Which region has the highest average order value?

---

## Tools Used

* SQL (MySQL/PostgreSQL)
* CSV dataset
* GitHub for version control

---

## Key Insights

* Electronics category generates the highest revenue.
* Top customers contribute a large share of total sales.
* Certain months show sales spikes.
* Some regions have higher average order value.
